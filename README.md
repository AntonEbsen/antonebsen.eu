# Anton Meier Ebsen Jørgensen — Institutional Digital Archive

A professional, institutional-grade digital portfolio for a European economist, specializing in monetary policy and quantitative analysis.

## Core Philosophical Pillars

* **Institutional Authority**: Designed with a "Master Scholar" aesthetic, inspired by the high-transparency publication standards of the ECB and IMF.
* **Privacy by Design**: Zero third-party trackers, cookies, or telemetry. All assets (fonts, scripts) are self-hosted to ensure data sovereignty.
* **Cognitive Accessibility**: Full adherence to WCAG 2.1 AA standards, featuring high-contrast themes, semantic HTML, and advanced typographic hyphenation.
* **Multilingual Integrity**: Full dynamic localization in English, Danish, German, and French.

## Technical Infrastructure

* **Architecture**: Optimized Vanilla HTML5 / CSS3 / JavaScript (ES6+).
* **PWA Readiness**: Integrated Service Worker for offline availability and institutional manifest for "Home Screen" installation.
* **Semantic Typography**: Custom implementations of 'Lora' and 'Inter' with optical alignment and hyphenation logic.
* **Bespoke Assets**: Hand-crafted SVG digital signatures and interactive scholarly components.

## Local Development

To review the institutional archive locally:

1. Clone the repository.
2. Run a local server (e.g., `npx serve .`).
3. Access via `localhost:3000`.

## Compliance & Standards

* **WCAG 2.1 AA**: AAA-ready contrast ratios and keyboard navigation.
* **GDPR**: Statistically zero data collection.
* **Mobile Excellence**: Fluid responsive layouts for policy review on any device.
